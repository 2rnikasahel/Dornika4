    15→ * Drizzle ORM schema for "درنیکا ساحل" (Dornika Sahel)
    16→ * luxury industrial e-commerce platform.
    17→ *
    18→ * The schema is shared between PostgreSQL (production) and PGlite (sandbox).
    19→ * All tables use text primary keys (cuid-style) to keep IDs portable.
    20→ *
    21→ * Column names use camelCase in TypeScript and snake_case in SQL.
    22→ * Indexes are named explicitly so they match between the runtime DDL
    23→ * (see `src/db/index.ts` BOOTSTRAP_SQL) and Drizzle migrations.
